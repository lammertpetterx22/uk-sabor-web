# Plugin Command

Lista y gestiona los MCP servers (plugins/skills) en Claude Code.

Ejecuta los siguientes comandos para mostrar información completa:

1. **Plugins instalados**: Lista los servidores MCP actualmente instalados
2. **Plugins disponibles**: Muestra un catálogo de todos los plugins que se pueden instalar
3. **Estado**: Verifica el estado de los servidores

Usa los comandos disponibles en el proyecto:
- `pnpm mcp:list` - Lista servidores instalados
- `cd skills && ./mcp-manager.sh available` - Muestra plugins disponibles para instalar
- `pnpm mcp:show` - Muestra configuración completa

Presenta la información de forma clara y organizada, agrupando los plugins disponibles por categoría.
