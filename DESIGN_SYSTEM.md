## Component Styling

### Buttons
- **Primary:** Gradient background (Magenta → Orange), white text, rounded corners
- **Secondary:** Transparent background with colored border, colored text
- **Hover:** Slight scale and shadow increase
- **Active:** Darker gradient

### Cards
- **Background:** #1A1A1A with subtle border
- **Border:** 1px solid rgba(255, 255, 255, 0.1)
- **Hover:** Slight lift effect (shadow increase)

### Forms
- **Input Background:** #2A2A2A
- **Input Border:** 1px solid rgba(255, 255, 255, 0.2)
- **Focus:** Border color changes to accent color
- **Label:** Small, uppercase, secondary text color

## Design Principles

1. **Vibrant & Energetic:** Use bold colors and dynamic layouts
2. **Accessible:** Maintain high contrast ratios (WCAG AA minimum)
3. **Modern:** Clean lines, generous spacing, smooth transitions
4. **Latin Dance Aesthetic:** Incorporate movement through animations and gradients
5. **Mobile-First:** Responsive design starting from small screens
6. **Performance:** Optimize images and minimize animations on slower devices

## Animation Guidelines

- **Transitions:** 200-300ms for UI interactions
- **Hover Effects:** Subtle scale (1.02-1.05) and shadow changes
- **Page Transitions:** Fade in (300ms) for smooth navigation
- **Loading States:** Spinner animations with accent color
